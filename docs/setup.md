# Setup

There are some scripts to help you prepare the setup for your interview, both if you are a candidate or an interviewer.

You can find the scripts you need in the `scripts` directory.

These scripts create a ready to use environment for your interview, like a fake `git` repository, a fake `cmake` project, etc.

- `git`: Scripts for git questions.
- `cmake`: Scripts for cmake questions.
- `cpp`: Scripts for cpp questions.

You can find more information about the specific scripts [here](scripts.md).
